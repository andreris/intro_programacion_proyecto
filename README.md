# Reto Memoria

Proyecto final de Introducción a la Programación: juego de memoria multijugador con tres niveles, siete rondas y dificultad progresiva.

## Cómo jugar

1. Abre `index.html`.
2. Selecciona uno de los tres niveles.
3. Registra entre uno y seis jugadores.
4. Memoriza la secuencia y escríbela en el mismo orden.
5. Cada respuesta correcta suma 10 puntos.

## Archivos

- `index.html`: interfaz visual y adaptable.
- `app.js`: experiencia de juego sin ventanas emergentes; utiliza funciones flecha y arreglos de objetos.
- `scriptFlechas.js`: script académico entregado con las funciones flecha para marcador y ganador.
- `scriptOriginal.js`: versión original conservada como referencia.

La interfaz permite elegir cada nivel, registrar jugadores, responder, revisar el marcador y conocer al ganador directamente en la página.
